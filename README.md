# Start

```bash
npm install

```

# Build

```bash
npm install
npm run prod
```

# Modifying React
The root of the React app is `App.jsx`. `RootComponents/Index.jsx` is the import statement, needed for server side rendering (SSR) to work. `RootComponents/Layout.jsx` is the root HTML layout for SSR to work.
